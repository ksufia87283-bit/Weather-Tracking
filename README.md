🌤️ Weather Tracker App
📌 Overview

This is a simple Weather Tracker desktop app built using Python. It allows users to enter a U.S. city and instantly get real-time weather information using the OpenWeatherMap API.

🚀 Features

Get current weather for any U.S. city

Displays:

City name

Weather condition

Temperature (°F)

Humidity (%)

Wind speed (mph)

Clean and simple user interface using Tkinter

Popup message display for results

🛠️ Technologies Used

Python

Tkinter (GUI)

Requests (API calls)

OpenWeatherMap API

⚙️ Setup Instructions

Install required library:

pip install requests

Get a free API key from:
https://openweathermap.org/api

Replace the API key in the code:

api = "your_api_key_here"

Run the program:

python your_file_name.py
🖥️ How to Use

Enter the name of a U.S. city

Click "Get weather"

A popup will display the weather details
